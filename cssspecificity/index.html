CSS Specificity สวัสดีชาวโลก

คือ ผมจะมาพูดถึง CSS Specificity กันว่ามันทำงานอย่างไร และมันคืออะไรกันนะครับ

CSS Specificity is fun

CSS Specificity คือ การให้ความสำคัญต่อ Selector ที่จะแสดงผลบนหน้าเว็บโดย web browser

เราจะมาพูดถึงคะแนนความสำคัญของค่า specificity กัน ถ้า selector ไหนมีคะแนนมากกว่าจะถูกเลือกแสดงผลบนเว็บ

    คะแนนความสำคัญของ selector
  • 1. การเขียนแบบ inline เช่นการเขียน css ใส่ใน style เลยจะได้คะแนน +1000
  • 2. การใช้ ID selector จะได้คะแนน +100
  • 3. การใช้ class selector จะได้คะแนน +10
  • 4. การใช้ element selector จะได้คะแนน +1

CSS Specificity จะให้คะแนนมากน้อยต่างกัน

Element Selector

ค่าคะแนนความสำคัญเท่ากับ +1

h1{

color: blue;

font-size: 18px;

}

ID Selector

ค่าคะแนนความสำคัญเท่ากับ +100

#top{

backgroud-color: pink;

padding: 18px;

}

inline

ค่าคะแนนความสำคัญเท่ากับ +1000

<a style="color: green">

Class Selector

ค่าคะแนนความสำคัญเท่ากับ +10

.left{

text-align: left;

padding: 18px;

}

    ถ้ามีการใช้งานของ selector ซ้อนกันหลายชั้น เราจะนำคะแนนของแต่ละ selector มารวมกันถ้า selector ตัวไหนคะแนนรวม มากสุดจะถูกแสดงผลบนเว็บเพจ
  • div a { } เท่ากับ 2 คะแนน
  • .top h1 { } เท่ากับ 11 คะแนน
  • div h3 a { } เท่ากับ 3 คะแนน
  • #top { } เท่ากับ 100 คะแนน
  • .top { } เท่ากับ 10 คะแนน

ดังนั้น selector #top จะถูกแสดงบนเว็บแทน

แต่ถ้าคะแนนเท่ากัน จะดูว่า Selector ตัวไหนที่ถูก import เข้ามาก่อนจะถูกนำมาใช้งานแทนนะครับ

สุดท้ายนี้ ถ้าผู้เขียนผิดพลาดประการใด ก็ขออภัยมา ณ ที่นี้นะครับ ขอบคุณครับ
